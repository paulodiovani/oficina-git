# Oficina git

## Comandos

* `git init`

  Inicializa o repositório

* `git add`

  Adiciona modificações para commit

* `git commit`

  Persiste as modificações no repositório

* `git status`

  Verifica pendências a commitar

* `git log`

  Visualiza histórico de modificações

* `git diff`

  Visualiza diferenças entre versões
